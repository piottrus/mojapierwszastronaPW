Możliwość formatowania tekstu w HTML

  • Początek i Zakończenie akapitu: < p > < / p >
  • Pogrubienie: < b > < / b >
  • Kursywa: < i > < / i >
  • Podkreślenie: < u > < / u >
  • Przekreślenie: < strike > < / strike >
  • Indeks górny: < sup > < / sup >
  • Indeks dolny: < sub > < / sub >
  • Przejscie do noweg wiersza: < br >
  • Spacja niełamliwa: & nbsp

Nagłówki w treści strony WWW

  1. < H1 > Będzie największym nagłówkiem
  2. < H2 > ... < H5 > Będzie coraz mniejsze
  3. < H6 > Będzie najmniejszym nagłówkiem

Możliwośći funckji font

  1. < font size="wielkość czcionki" >
  2. < font color="kolor czcionki" >
  3. < font face="rodzaj czcionki" >

Najprostsze rodzaje kolorów do użycia w funkcji font

  1. Nazwą: red-czerwony, green-zielony, blue-niebieski
  2. RGB: rgb(255,0,0)-czerwony, rgb(0,255,0)-zielony, rgb(0,0,255)-niebieski
  3. Hexadecymalny, # RR GG BB : #FF0000-czerwony, #00FF00-zielony, #0000FF-niebieski
